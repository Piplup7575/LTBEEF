LTBEEF

- Drag the button above to your bookmarks bar
- Visit chrome://extensions, select the extention you want to disable, and copy the ID of the extension you want to disable. (or use one of the common ids below)
- Visit https://chrome.google.com/webstorex. (This is a 404 page, and that is ok. If it is blocked, join the TN Discord server for support!)
- Click the bookmark (NOTE: This only works on the Chrome Web Store.)
- Paste the extension ID in the prompt.
You're done! The extention should now be disabled.
